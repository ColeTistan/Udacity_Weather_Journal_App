# Udacity-Weather-Jouranl-Page
The purpose of this project is to integrate an existing API using asynchronous. Several tasks for this project included the following: dynamically appending data to the DOM, show how JavaScript can improve the usability of an static site and construct helper functions to make readable code.

## Technologies Used
`HTML5`
`CSS3`
`ES6 JavaScript`
`OpenWeather API`

## Instructions
1) Download zip file or clone from this repository.
2) Open project folder and run project using this command in the terminal `node server.js`
3) Type in a zip code (Ex: 10001) and type how you are feeling (Ex: Lovely day).

## Notes
1) You can run this project on any browser of your choice.
2) Several helper functions were added to make code readable. 
   - `convertImperial` & `convertMetric` functions get the standard temperature from the API and convert it to both Imperial and Metric.
   - `dateBuilder` function constructs the current date.
